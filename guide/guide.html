클래스명 규칙

  1. 소문자만 작성 가능
  2. 단어와 단어 사이에는 -로 구분
  3. ex) tit-area

css 작성규칙 규칙

  1. 속성과 속성값 사이는 띄어쓰기 없고 (;) 뒤에는 1칸 띄어쓰기{속성명:속성값; 속성명:속성값;}
  2. 마지막 속성도 반드시 (;)로 끝내기
  3. 다중 선택자 선언시 (,) 후 줄바꿈 하기
    img,
    input {font-size:1.6rem;}

스크립트 규칙

  1. 변수명/함수명 명사로 시작
  2. 낙타기법 사용
  3. ex) currentIdx

이미지명 규칙

  1. 소문자만 작성 가능
  2. 스네이크기법 사용
  3. ex) bg_juju.jpg, img_juju.jpg
  4. 이미지명 시작하는 단어
    백그라운드 :bg_
    버튼 :btn_
    아이콘 :ico_
    이미지 :img_

주석 규칙

  1. 여기서 s는 콘텐츠의 시작 start e는 콘텐츠의 끝 end
  2. 기본 작업 주석
    html
    <!-- 콘텐츠명 (s) -->
    <!-- 콘텐츠명 (e) -->
    css
    큰 구조의 주석
    /*=====분류 콘텐츠명=====*/ (선택)
    /* 콘텐츠명 */
    script
    //콘텐츠명(s)
    //콘텐츠명(e)
  3. 수정 작업 주석(완료되고 작업할 때)
    html
    한줄일 때 (html 마크업 뒤에 주석 달기)
    <!-- 수정 240419 박소현: 제목변경 -->
    <!-- 수정 240419 박소현 레이웃변경 (s) -->
    <!-- 수정 240419 박소현 레이웃변경 (e) -->
    css {} 괄호 뒤에 주석 달기
    /* 수정 240419 박소현 컬러변경 */
    script 한줄 일 때
    //수정 240419 함수명 변경
    script 두줄이상 일 때
    //수정 240419 박소현 스크립트 전체 변경(s)
    //수정 240419 박소현 스크립트 전체 변경(e)
